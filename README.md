
## Static Translations for any front end project.


- Start index.js 
- Parse target folder (src) for all <t> tags.
- Build JSON file of translations from English
- Copy src/ to dist/{lang}/
- Replace each dist/{lang}/ with their translations.



`[
  {
    file: '/asdasda/asdsa/',
    en: 'original string',
    fr: 'originale cord
  }
]`
